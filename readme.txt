# Проект: Работа с процессами и переменными окружения

## Описание
Данный проект демонстрирует работу с процессами в операционной системе Linux, а также взаимодействие между родительским и дочерним процессами. Родительская программа (parent.c) создает дочерние процессы, которые запускают другую программу (child.c). Дочерняя программа читает файл env.txt и выводит значения переменных окружения, указанных в этом файле.

---

## Состав проекта
- `parent.c` — родительская программа, которая создает дочерние процессы и передает им управление.
- `child.c` — дочерняя программа, которая читает файл env.txt и выводит значения переменных окружения.
- `env.txt` — текстовый файл, содержащий список переменных окружения для вывода.
- `Makefile` — файл для автоматизации сборки проекта.
- `build/` — директория, в которой находятся скомпилированные исполняемые файлы.

---

## Требования
- Операционная система: Linux (Ubuntu или другая совместимая).
- Компилятор: GCC.
- Установленные утилиты make.

---

## Сборка проекта
Для сборки проекта используется Makefile. Выполните следующую команду в терминале:

make MODE=release
