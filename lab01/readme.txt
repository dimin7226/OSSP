/*
 * Программа dirwalk
 *
 * usage: ./dirwalk [options] [dir] [options]
 *
 * Опции:
 *   -l     вывод только символических ссылок
 *   -d     вывод только каталогов
 *   -f     вывод только обычных файлов
 *   -s     сортировать выход в соответствии с LC_COLLATE
 *
 * Если опции l, d и f опущены, выводятся все типы (каталоги, файлы и ссылки).
 *
 * Формат вывода аналогичен утилите find: выводятся пути к найденным файлам и каталогам.
 *
 * Программа переносима (используются только возможности POSIX).
 */
 
 1. После раскаповки архива, перейдите в директорию lab01:
 cd Чехович С.Д./lab01
 2. Соберите проект:
 make
    Чтобы собрать в режиме отладки:
 make MODE=debug
    Чтобы собрать в режиме выпуска:
 make MODE=release
 3. Скомпилируйте проект:
    В режиме отладки:
 gcc -g -ggdb -std=c11 -pedantic -W -Wall -Wextra src/main.c -o dirwalk
    В режиме выпуска:
 gcc -std=c11 -pedantic -W -Wall -Wextra -Wno-unused-parameter -Werror src/main.c -o dirwalk
 4. Используйте dirwalk.
